Detailed Description

template<class MultiAgentSolver>
class IndependenceDetection< MultiAgentSolver >

Independence Detection search.

Enhanced version of ID (also called EID) : tries to resolve a conflict (by attempting to replan one group to avoid the plan of the other group) before merging the groups

Only takes in account negative constraints of problem

Variable low-level search algorithm Toggelable CAT enhancement
